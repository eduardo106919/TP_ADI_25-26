## 📘 Descrição Geral

Este projeto foca-se na conceção e desenvolvimento de modelos de aprendizagem e decisão, utilizando a plataforma **KNIME**. O trabalho envolve a análise de dados e extração de conhecimento, dividindo-se em duas tarefas desenvolvidas em simultâneo:

1. **Dataset Grupo:** Seleção de um dataset de escolha livre (em plataformas como Kaggle ou UCI) para exploração, preparação dos dados e desenvolvimento de diversos modelos de *machine learning*.
2. **Dataset Atribuído:** Análise e tratamento de um dataset previamente distribuído pelos docentes, focado na conceção de modelos de classificação e regressão.

Em ambas as tarefas, procede-se à extração de conhecimento e é realizada uma análise crítica detalhada dos resultados obtidos.

## 🎯 Objetivos

Com a realização deste Trabalho de Grupo pretende-se sensibilizar e motivar os estudantes para a conceção e desenvolvimento de um projeto prático, no âmbito da unidade curricular de Aprendizagem e Decisão Inteligentes, utilizando diferentes paradigmas de aprendizagem e recorrendo a técnicas de **machine learning** abordados no decurso da unidade curricular.
